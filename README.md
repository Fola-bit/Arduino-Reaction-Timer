# Arduino Reaction Timer Game

An interactive Arduino game with LEDs, buzzer, and pushbutton measuring reaction time.  
Includes countdown animation, LED feedback for reaction speed, and false start detection.

## How It Works
- Countdown LED animation
- Random LED lights + buzzer
- Button press measures reaction time
- Fastest reaction tracked
- LED feedback shows slow, average, or fast reaction

## Components
- Arduino Uno R3
- 3 LEDs + resistors
- Pushbutton switch
- Piezo buzzer
- Breadboard + jumper wires
